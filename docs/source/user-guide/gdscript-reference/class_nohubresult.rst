NohubResult
===========

**Inherits:** ``RefCounted``

**Inherited By:** :ref:`NohubResult.Address<class_NohubResult_Address>`, :ref:`NohubResult.Lobby<class_NohubResult_Lobby>`, :ref:`NohubResult.LobbyList<class_NohubResult_LobbyList>`

Represents the result of a :ref:`NohubClient<class_NohubClient>` call

.. rst-class:: classref-introduction-group

Description
-----------

If the call was successful, this instance will store the resulting value, and :ref:`is_success()<class_NohubResult_method_is_success>` will return ``true``. This class has multiple specializations, each implementing a ``value()`` method that returns a type-safe success result. 



If the call failed, :ref:`is_success()<class_NohubResult_method_is_success>` will return ``false``. The actual error is returned by :ref:`error()<class_NohubResult_method_error>`.
